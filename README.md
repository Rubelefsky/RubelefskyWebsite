# Rubelefsky Website

A professional portfolio website showcasing IT Support and Cybersecurity expertise.

## Technologies Used

- **HTML5** - Semantic markup for content structure
- **CSS3** - Custom styling with modern features
- **JavaScript (Vanilla)** - Interactive features and animations
- **Google Analytics** - Website traffic tracking and analytics
- **Intersection Observer API** - Scroll-based animations

## Project Structure

```
RubelefskyWebsite/
├── .github/              # GitHub configuration and workflows
├── .git/                 # Git version control
├── mainpage/            # Main website files
│   ├── index.html       # Main HTML file with portfolio content
│   ├── styles.css       # Custom CSS styling
│   ├── script.js        # JavaScript for smooth scrolling and animations
│   ├── robots.txt       # SEO crawler instructions
│   └── mainpage.md      # Additional documentation
└── README.md            # This file
```

## Features

- **Responsive Design** - Mobile-friendly layout
- **Smooth Scrolling** - Navigation with smooth scroll behavior
- **Scroll Animations** - Elements fade in on scroll using Intersection Observer
- **Dynamic Header** - Header styling changes on scroll
- **SEO Optimized** - Includes robots.txt for search engine crawlers

## Sections

- **Hero** - Introduction and call-to-action
- **About** - Professional background and skills
- **Experience** - Career timeline and goals
- **Contact** - Links to LinkedIn, GitHub, and email
